Umur serta size ikan, apa apabila kita meminati ikan koi yang berusia 1 tahun, ukuran relatif cukup besar serta di atas rata-rata merupakan pilihan (jumbo tosai) ukuran type ini banyak diminati beberapa pemelihara\/pencinta ikan koi ( favorit ), semakin besar semakin bernilai tinggi pastinya.<\/p>\n\n\n\n

Saat type (varietas), kualitas, dan ukuran ikan koi sudah dipilih, maka penerapan koi pond atau kolam koi juga amat memastikan. ukuran kolam koi tidak sama \u2013 lain sesuai dengan kondisi luas area dimana kolam koi ini dapat diletakkan sesuai dengan wujud serta estetika pastinya.<\/p>\n\n\n\n
